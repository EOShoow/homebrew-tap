# homebrew-tap

Homebrew formulas for EOShoow tools.

## Codex Battery

A tiny macOS menu bar battery for Codex quota.

```bash
brew install EOShoow/tap/codex-battery
codex-battery
```

Optional login startup:

```bash
codex-battery-login install
```

Remove login startup:

```bash
codex-battery-login uninstall
```

Project: <https://github.com/EOShoow/codex-battery>
